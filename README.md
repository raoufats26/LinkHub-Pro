# 🚀 LinkHub Pro

A modern **link-in-bio SaaS platform** with analytics and customization.

## ✨ Features

- 🔐 Authentication system
- 📊 Analytics dashboard (Chart.js)
- 🔄 Drag & drop link management
- 🌍 Public profile pages
- 🎨 Modern UI (TailwindCSS)

## 🛠 Tech Stack

- Flask
- PostgreSQL
- TailwindCSS
- Chart.js
- Sortable.js
